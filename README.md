## Installation

You can install the development version of gdiviz from
[GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("ksreyes/gdiviz")
```

## Example

``` r
library(gdiviz)
gdiplot(code = "stocks", country = "DEU")
```
